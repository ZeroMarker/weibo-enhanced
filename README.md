# 微博增强版 (Weibo Enhanced)

第三方微博网页应用，提供更好的用户体验和UI增强。

## 功能特性

- 🎨 现代化UI设计
- 🌙 深色模式支持
- 📱 响应式布局
- ⚡ 更快的加载速度
- 🔍 增强的搜索功能
- 📊 数据统计可视化

## 技术栈

- **前端框架**: Next.js 14
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **状态管理**: React Hooks
- **数据获取**: SWR

## 快速开始

### 安装依赖

```bash
npm install
# 或
yarn install
# 或
pnpm install
```

### 开发模式

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash
npm run build
npm run start
```
